# Best_deals_forever

A basic blog app made in flutter with firebase as backend.

## Used following flutter packages:-
* cloud_firestore: ^0.13.5
* flutter_html: ^0.11.1
* url_launcher: ^5.4.5
* social_media_buttons: ^0.1.2
* firebase_admob: ^0.9.0+1
